In order to continue with this project by building the ALU, I needed only to follow the instructions provided and follow along with the instructional videos for this project by Ben Eater, an electrical engineering educator online. My process would be to first watch the videos to get a broad overview of the building process and learn about how the different components work together, then assemble the project on my own using the high level circuit diagram for the ALU and looking at the datasheets for the individual chips. Finally, once I have built the module and made sure that it works, I will rewatch the instructional videos to solidify my understanding of how the ALU functions at a fundamental level.

WHAT?

Building the Arithmetic Logic Unit for this computer was relatively straightforward as I have already built an adder using digital logic for another project that I worked on. By constructing single bit adders that are cascaded together with carry in and carry out bits, it's relatively straightforward to create the adder. What was new to me was how to perform subtraction with two's complement. By converting the number that you're subtracting into a two's complement number, you can then add it to your other input and it will produce the correct subtraction output. To convert to two's complement, I used XOR gates from all of the B register inputs so that the bits would invert when a subtraction pin was HIGH. This bit would also be pushed to the carry in input of the least significant bit to complete the two's complement number. Finally, the output of the ALU was added to the bus through a tristate buffer, allowing the computer to control when it should activate.
