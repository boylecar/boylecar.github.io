WHAT?

At a fundamental level, an 8-bit register is made from 8 D-flip-flops. On the rising edge of each clock signal, a D-flip-flop will update its output based on its input and store that value in "memory." By wiring 8 of these flip flops in parallel, you create a register. With a control pin, you can select whether the flip-flop will read the input and push it to its outputs or if it will keep its previous state. By connecting the output of these registers through LED's, it's easy to see visually the data stored in the register. Additionally, the output of the register is put through a tristate buffer, which allows data to flow only when its enable pin is HIGH. From there, the data is output onto the common bus whenever the enable pin is HIGH. With the CLK, Read, and Enable pins, you have control over the register and it can be effectively implemented with the rest of the computer.
