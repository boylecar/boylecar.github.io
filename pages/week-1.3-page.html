Week 1 - Arduino EEPROM Programmer

WHY?

As would be expected due to my choice of major, I am highly interested in understanding any system or product that uses electricity. The most important and prevalent today broadly fits under the category of "computer." In my pursuit of a deeper understanding of electronics and more specifically computers, I started off this project with the goal of gaining more knowledge about the fundamentals of a computer. The best way for me to really understand such a complex system is to start building it myself. The goal I defined clearly for this week is:

Continue building a simple computer by understanding its components at a fundamental level.

HOW?

Now that my computer is starting to come together, the next piece that would be important is to make it possible to output user-friendly information so that it doesn't require specialized knowledge of each peace to understand. To do this, I will use four 7-segment decimal displays. Three of them will be used to represent a value up to 255, and the fourth will represent a positive or negative sign value. In order to do build the logic needed to control the values for each of the segments of the display for each value, I will use a ROM chip to replace combinational logic. This week, I will build a robust EEPROM programmer using an Arduino Nano, so that I can simply upload a program to the Arduino that will change the information stored in the EEPROM.

WHAT?

In order to achieve the effect that I want where some 8-bit input will be interpreted by the EEPROM to output the decoded values for the individual segments of the 4 displays, I will essentially be programming the output values into the memory of the EEPROM, stored at the address that is specified by the inputs. For example, if 142 should be interpreted by the EEPROM and output a binary value of 11000101, then I will program the EEPROM to have the value 11000101 at the address 142. Because the Arduino doesn't have enough pins for all 11 inputs and 8 outputs, however, I will build an 8-bit shift register with two IC chips that will be used to set the value at the register specified by the Arduino. In practice, the Arduino will shift out 8 bits of information, set the 11 address pins to the correct value, and enable the EEPROM to reprogram. Next week, I can use this EEPROM to replace the combinational logic needed in the output register of my computer.
