HOW?

This week I will install the final hardware to complete my computer. This week I learned about what it means for a computer to be "computationally complete." Formally, this means that the computer can compute anything that a Turing Machine can compute. This is an entirely too large of a topic to describe fully, but it is important to know that the only thing missing for my computer to be Turing complete is the ability to perform if, then operations. To do this, I build a Flags Register that will store data when a specific signal goes HIGH. The signals I will use are both derived from the ALU. The first flag is the Zero Flag which is raised when the value in the ALU is zero. The second flag is the Carry Flag, which goes HIGH when the ALU has a carry-out bit. Both of these will be driven as inputs to the EEPROM, and used in if, then statements. For example, I could perform subtraction between the A and B register. If A and B are equal, the value stored in the ALU is zero and the Zero Flag is raised. I can use this to represent the expression: "if A == 5: "

WHAT?

To implement the Flags Register, I use the same flip-flop chip that I have used for previous modules to store the values of the Carry and Zero Flags into the Flags register. The enable signal which updates the values stored in this register is also implemented into the CPU, so that during a program it can store the state of the Flags into the Flags Register. On the next step of the program, it could use these values and do something different depending on the different states of the Flags. This essentially creates a feedback loop, where the output of a program is fed back into the input of the CPU which executes the program itself. Finally, I will reprogram the EEPROM's to account for these inputs.
